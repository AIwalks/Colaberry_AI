## Why These Files Are Here

These 29 files were moved here from `/integration/` and `/implementation/` during the Sprint 0 repository cleanup.

They were archived rather than deleted because they contain some valid philosophical intent (human oversight, governance visibility, operational safety) buried under enterprise-governance template language that was not actionable for this system's actual scope.

**None of these files describe how the system actually works.** They describe aspirational properties using a 22-section template structure. For the system as it actually exists, see the `/audit/` directory.
